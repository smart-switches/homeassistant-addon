# smart-switches

A Home Assistant Add-On for managing smart lightswitches!

Consists of a frontend site, and backend server

## Development

**Backend:**
```bash
make run-server  # Builds and runs server on :8124
```

**Frontend:**
```bash
make run-site    # Starts Gatsby on :7128
```

Open http://localhost:7128 - API calls proxy to backend automatically.
